# Code of Conduct

This project is intended to be a respectful, inclusive, and constructive community.

## Expected behavior
- be kind and professional
- welcome diverse perspectives
- focus on ideas and improvement

## Unacceptable behavior
- harassment or discrimination
- personal attacks
- disruptive or abusive conduct

## Enforcement
Community members who violate this code may be removed from participation at the discretion of the maintainers.
