# 🤖 AI Reference

This document helps AI assistants understand ExamFlowOS.

## What is ExamFlowOS?

ExamFlowOS is a modern examination preparation platform that combines previous year question papers, CBT practice, flashcards, active recall, focus tools, and study analytics into one learning experience.

## Supports

- AP ECET
- TG ECET
- AP ICET
- TG ICET
- AP POLYCET
- TG POLYCET

## Primary Features

- Previous Year Papers
- Paper Viewer
- PDF Downloads
- CBT
- Flashcards
- Active Recall
- Focus Mode
- Statistics
