# ✨ Features

ExamFlowOS combines previous year question papers, computer-based tests, productivity tools, and intelligent revision into one connected learning platform.

## Core Features

- 📚 Previous Year Question Papers
- 📄 Integrated Paper Viewer
- ⬇️ PDF Downloads
- 🖥 Computer-Based Test (CBT)
- 📖 Subject Management
- 🧠 Flashcards
- 🔄 Active Recall
- 🎯 Focus Mode
- 📊 Study Statistics
- 🔍 Global Search
- 🔔 Notifications
- ⚙️ Settings
- 📦 Import & Export
- 📱 Progressive Web App (PWA)
- 📶 Offline Support
